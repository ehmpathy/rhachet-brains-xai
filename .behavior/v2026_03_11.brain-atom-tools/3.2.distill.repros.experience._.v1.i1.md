# distill: experience reproductions

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| tool invocation with output schema | `atom.ask()` | pass plugs.tools + schema.output | result.calls.tools populated, result.output null | integration |
| tool invocation without output schema | `atom.ask()` | pass plugs.tools, no response_format | result.calls.tools populated, result.output null | integration |

## reproduction feasibility

### experience: tool invocation with output schema

**utilities**: test-fns (given/when/then/useThen), zod, jest

**setup**:
```ts
const toolDef = {
  slug: 'weather.lookup',
  description: 'get weather for city',
  schema: { input: z.object({ city: z.string() }) },
};
```

**test sketch**:
```ts
given('[case1] brain with tools + output schema', () => {
  when('[t0] ask requires tool', () => {
    const result = useThen('it returns tool call', async () =>
      atom.ask({
        plugs: { tools: [toolDef] },
        prompt: 'what is the weather in austin?',
        schema: { output: outputSchema },
      }),
    );

    then('calls.tools is populated', () => {
      expect(result.calls?.tools).toBeDefined();
      expect(result.calls?.tools.length).toBeGreaterThan(0);
    });

    then('output is null', () => {
      expect(result.output).toBeNull();
    });
  });
});
```

### experience: tool invocation without output schema

**same as above**, but implementation does not send `response_format` to xai.

### experience: brain answers directly

**test sketch**:
```ts
when('[t1] ask does not require tool', () => {
  const result = useThen('it returns direct answer', async () =>
    atom.ask({
      plugs: { tools: [toolDef] },
      prompt: 'say hello',
      schema: { output: outputSchema },
    }),
  );

  then('calls is null', () => {
    expect(result.calls).toBeNull();
  });

  then('output is populated', () => {
    expect(result.output).toBeDefined();
  });
});
```
