### round 3: critical contract discovery

after reading the actual rhachet types, i found:

1. **BrainAtom.ask() REQUIRES `schema: { output }`** — it is NOT optional
2. **the wish says "w/o output schema"** — this CANNOT mean "without schema parameter"

**interpretation**: "w/o output schema" means:
- do NOT send `response_format: json_schema` to xai
- still pass `schema: { output }` to satisfy rhachet contract
- xai returns free-form text, we parse it against the zod schema

**w/ output schema** means:
- send `response_format: json_schema` to xai (structured output)
- xai validates output format at provider level

this is a CRITICAL question added to the vision for wisher confirmation.
