## test rule: useThen for shared results

**rule says:** use useThen to share API results across then blocks. avoid redundant calls.

**blueprint test:** multiple assertions on same ask() result (calls.tools, output, slug, input, exid).

**why this holds:** the pattern requires:
```ts
const result = useThen('ask succeeds', async () => brainAtom.ask(...));
then('calls.tools populated', () => expect(result.calls?.tools).toBeDefined());
then('output is null', () => expect(result.output).toBeNull());
```

one API call, multiple assertions. this is the useThen pattern.

**verdict:** ADHERES
