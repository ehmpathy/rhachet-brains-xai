# 5.2.evaluation: credential supplier context

## summary (as implemented)

implemented context-injected credential supplier for `genBrainAtom`:
- accept `ContextBrainSupplier<'xai', BrainSuppliesXai>` via context arg to `ask()`
- extracted `getSdkXaiCreds` as domain operation for credential resolution
- getter pattern: `() => Promise<{ XAI_API_KEY: string }>`
- precedence: creds getter > env var fallback
- exported `BrainSuppliesXai` type for consumers
